Objetos em Javascript

  • Declaração: var variavel = {atributo:"elemento", nome:"Paulo",idade:44, atualizar(anos){this.idade += anos} }
  • Pode ser declarada uma função como elemento do objeto
  • Se declarado variavel.atualizar(3), a idade passa a ser 47
  • Para incluir novo atributo: variavel.peso = 70
  • Para excluir um atributo: delete variavel.atributo
  • Para incluir um array: variavel.hobbies = ["xadrez","boliche"]
  • Para "chamar" o elemento: variavel.atributo // retorna o elemento
  • Também pode ser feito variavel["atributo"] que também retorna o elemento
  • Forma tem mais utilidade quando um atributo possuir um caractere especial. Exemplo: variavel.["last-name"]
  • var cor = "azul" // var decoracao = {medida:"grande",cor } // Quando a variável é declarada anteriormente, é necessário informar apenas a variável, que ela já assume variável e valor. Equivale a: var decoracao = {medida:"grande",cor:cor }. Isso se o nome da propriedade for o mesmo nome da variável declarada
  • Juntando dois objetos: Object.assign (objeto alvo, outros objetos). Se o objeto alvo for informado como {}, será criado um novo objeto.
